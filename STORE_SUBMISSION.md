# Chrome Web Store Submission Checklist

## Ready

- Extension name matches listing: `Tab Audio Normalizer`
- Icons present: 16, 32, 48, 128
- No remote code loading
- Audio processing stays local
- Privacy policy included

## Before Upload

- Create a release ZIP from the extension root
- Verify `manifest.json` has the final version number
- Add store screenshots and listing text
- Confirm permissions explanation in the Web Store form

## Review

- Test install in a clean Chrome profile
- Test enable, disable, reload, and new tab behavior
- Check console for errors on common sites
